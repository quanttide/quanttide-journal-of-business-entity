## OWL 交付物规范

### 一、命名空间

```
@prefix qt:   <http://quanttide.com/ontology/>
@prefix rdf:  <http://www.w3.org/1999/02/22-rdf-syntax-ns#>
@prefix owl:  <http://www.w3.org/2002/07/owl#>
@prefix rdfs: <http://www.w3.org/2000/01/rdf-schema#>
```

### 二、类定义 (Classes)

| 类名 | 说明 | 示例 |
|-----|------|------|
| qt:BusinessStage | 业务流程阶段 | 需求定义阶段、商务洽谈阶段 |
| qt:Process | 具体流程 | 需求受理流程、报价审批流程 |
| qt:ProcessStep | 流程步骤 | 客户发起需求、需求初探 |
| qt:BusinessRule | 业务规则 | 报价浮动规则、数据保留规则 |
| qt:TriggerCondition | 触发条件 | 关键词数量变化、客户要求按年份统计 |
| qt:SOP | 标准作业程序 | SOP-01 需求调研规范 |

### 三、对象属性 (Object Properties)

| 属性名 | 域 | 值域 | 说明 |
|-------|-----|-----|------|
| qt:hasStage | qt:Process | qt:BusinessStage | 流程所属阶段 |
| qt:hasStep | qt:Process | qt:ProcessStep | 流程包含的步骤 |
| qt:precedes | qt:ProcessStep | qt:ProcessStep | 步骤先后关系 |
| qt:hasRule | qt:BusinessStage | qt:BusinessRule | 阶段包含的业务规则 |
| qt:triggers | qt:TriggerCondition | qt:BusinessRule | 触发条件触发业务规则 |
| qt:hasSOP | qt:BusinessStage | qt:SOP | 阶段对应的 SOP |

### 四、数据属性 (Data Properties)

| 属性名 | 域 | 值域 | 说明 |
|-------|-----|-----|------|
| qt:hasFormula | qt:BusinessRule | xsd:string | 规则公式 |
| qt:hasRetentionPeriod | qt:BusinessRule | xsd:integer | 保留期限（天） |
| qt:hasDescription | owl:Thing | xsd:string | 描述信息 |

### 五、核心本体结构

```
qt:BusinessStage
  ├── qt:需求定义阶段
  │   ├── qt:hasProcess → qt:需求受理流程, qt:需求变更管理流程
  │   ├── qt:hasRule → qt:报价浮动规则
  │   └── qt:hasSOP → qt:SOP-01, qt:SOP-02
  ├── qt:商务洽谈阶段
  │   ├── qt:hasProcess → qt:范围界定流程, qt:报价审批流程
  │   └── qt:hasSOP → qt:SOP-02
  ├── qt:数据交付阶段
  │   ├── qt:hasProcess → qt:数据生产流程, qt:交付物验收流程
  │   └── qt:hasSOP → qt:SOP-03
  └── qt:运维与售后阶段
      ├── qt:hasProcess → qt:历史数据追溯流程, qt:争议处理流程
      └── qt:hasSOP → qt:SOP-04

qt:ProcessStep (示例：需求变更管理流程)
  qt:需求确认 → qt:precedes → qt:变更请求
  qt:变更请求 → qt:precedes → qt:变更评估
  qt:变更评估 → qt:precedes → qt:变更确认
  qt:变更确认 → qt:precedes → qt:需求冻结

qt:BusinessRule (示例)
  qt:报价浮动规则
    qt:hasTrigger → "关键词数量或采集范围发生变化"
    qt:hasFormula → "报价 = (基础启动费 + 单条数据成本 × 数量) × 复杂度系数"
  qt:数据保留与归档规则
    qt:hasRetentionPeriod → 30
  qt:数据统计粒度规则
    qt:hasDescription → "非结构化数据不支持按年份统计"
```

        </qt:hasFormula>
    </qt:BusinessRule>
